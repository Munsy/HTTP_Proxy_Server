HOW TO COMPILE OUR PROGRAM:

Regardless of the operating system you are using, you need to set your proxy settings for your browser to run through 127.0.0.1 on port 45500.

If using Windows (Not recommended):
	- Simply open the solution in Visual Studio 2015 and run from there.

If using Linux:
	- Run the 'mk' script.

If you experience any problems, contact tmunson@eecs.wsu.edu
